# WebsiteTcarvi

- Versões:
  - NodeJs: 22.12.0
  - npm: 10.9.0
  - npx: 10.9.2
  - Angular CLI: 19.0.4


## Comandos para executar website

```bash
npm install
```
```bash
npm build
```
```bash
npm run start 
```

## Comandos para manutenção do website
```bash
ng generate --help
```  
```bash
ng generate component component-name -d
```  
```bash
ng g c layout
```  


```bash
ng test
```

```bash
ng e2e
```
